Kalkulator Time Value of Money

by masmasdefault

1. Future Value (FV) dari Present Value (PV)

Contoh Soal:

Anda memiliki uang Rp 1.000.000 dan disimpan di bank dengan bunga 10% per tahun selama 5 tahun. Berapakah nilai uang tersebut 5 tahun yang akan datang?

PV = Rp 1.000.000
r = 10% = 0,10
n = 5 tahun

FV = PV(1 + r)n = 1.000.000(1 + 0,10)5 = Rp 1.610.510

2. Present Value (PV) dari Future Value (FV)

Contoh Soal:

Anda ingin memiliki uang Rp 5.000.000 lima tahun dari sekarang. Jika suku bunga 12% per tahun, berapa uang yang harus Anda tabung sekarang?

FV = Rp 5.000.000
r = 12% = 0,12
n = 5 tahun

PV = FV / (1 + r)n = 5.000.000 / (1 + 0,12)5 = Rp 2.832.710

3. Suku Bunga (r) dari PV dan FV

Contoh Soal:

Anda memiliki tabungan Rp 2.000.000 dan dalam 3 tahun menjadi Rp 2.500.000. Berapakah suku bunga per tahun?

PV = Rp 2.000.000
FV = Rp 2.500.000
n = 3 tahun

FV = PV(1 + r)n
2.500.000 = 2.000.000(1 + r)3
(1 + r)3 = 2.500.000 / 2.000.000 = 1,25
1 + r = 1,251/3 = 1,0772
r = 0,0772 = 7,72%

4. Jumlah Tahun (n) dari PV dan FV

Contoh Soal:

Anda memiliki tabungan Rp 1.500.000 dan ingin menjadi Rp 2.000.000 dengan suku bunga 8% per tahun. Berapa tahun yang dibutuhkan?

PV = Rp 1.500.000
FV = Rp 2.000.000
r = 8% = 0,08

FV = PV(1 + r)n
2.000.000 = 1.500.000(1 + 0,08)n
2.000.000 / 1.500.000 = (1,08)n
1,3333 = (1,08)n
ln(1,3333) = n * ln(1,08)
n = ln(1,3333) / ln(1,08) = 3,7 tahun

5. Future Value Anuitas (FVA)

Contoh Soal:

Anda menabung Rp 2.000.000 setiap tahun selama 10 tahun dengan suku bunga 7% per tahun. Berapa total tabungan Anda setelah 10 tahun?

PMT = Rp 2.000.000
r = 7% = 0,07
n = 10 tahun

FVA = PMT * [((1 + r)n - 1) / r]
FVA = 2.000.000 * [((1 + 0,07)10 - 1) / 0,07] = Rp 27.836.534

6. Present Value Anuitas (PVA)

Contoh Soal:

Anda berencana menabung Rp 1.000.000 setiap tahun selama 5 tahun. Jika suku bunga 8% per tahun, berapakah nilai sekarang dari rangkaian tabungan tersebut?

PMT = Rp 1.000.000
r = 8% = 0,08
n = 5 tahun

PVA = PMT * [(1 - (1 + r)-n) / r]
PVA = 1.000.000 * [(1 - (1 + 0,08)-5) / 0,08] = Rp 3.992.710

7. Pembayaran Anuitas (PMT)

Contoh Soal:

Anda memiliki utang Rp 50.000.000 yang harus dilunasi dalam 5 tahun dengan bunga 10% per tahun. Berapa pembayaran tahunan yang harus Anda lakukan?

PVA = Rp 50.000.000
r = 10% = 0,10
n = 5 tahun

PVA = PMT * [(1 - (1 + r)-n) / r]
50.000.000 = PMT * [(1 - (1 + 0,10)-5) / 0,10]
PMT = 50.000.000 / [(1 - (1 + 0,10)-5) / 0,10] = Rp 13.180.592
